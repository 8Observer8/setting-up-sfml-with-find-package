# setting-up-sfml-with-find-package

A project template for building and running an SFML 2.6.2 application on Windows using MinGW 13.1 (WinLibs) and CMake.

## 📖 Full Tutorial
For the complete step-by-step instructions on setting up the environment, configuring the system PATH, and opening the project in different IDEs, follow my guide here:

**[SFML Desktop: Setup with find_package (MinGW 13.1)](https://8observer8.github.io/tutorials/sfml2/setting-up-sfml2/?lang=en)**

## 🚀 Quick Start
If you already have your environment configured:

1. Clone the repository.
2. Ensure SFML 2.6.2 is at `C:/libs/SFML-2.6.2-mingw`.
3. Use the provided automation scripts:
   - `config-exe.bat`
   - `build-exe.bat`
   - `run-exe.bat`
